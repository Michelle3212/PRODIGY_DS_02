# PRODIGY_DS_02
## Task-02: Data Cleaning and Exploratory Data Analysis (EDA)

This project involves data cleaning and exploratory data analysis using the Titanic dataset.

### 📊 Dataset:
- Titanic Dataset

### 🔧 Tools Used:
- Python
- Pandas
- Matplotlib
- Seaborn

### 🛠️ Work Done:
- Handled missing values in Age and Fare columns
- Removed irrelevant columns such as Cabin
- Performed exploratory data analysis using visualizations
- Identified patterns and relationships between variables

### 📌 Observations:
- Most passengers belong to 3rd class  
- Males are more than females  
- Most passengers are aged between 20–40  
- Fare is highly skewed (few paid very high)  
- Higher class → higher fare  

### 📷 Output:
- Count plots for gender and class
- Histograms for age and fare distribution
- Boxplots showing relationship between class and fare
- Correlation heatmap

### 🎯 Learning Outcome:
This task helped in understanding:
- Data preprocessing techniques
- Handling missing data
- Performing EDA using visualization tools
- Extracting meaningful insights from data
